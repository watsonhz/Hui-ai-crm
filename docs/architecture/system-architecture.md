# AI-CRM 企业级政企大B客户管理系统架构设计 (v4.0)

## 1. 整体架构
采用前后端分离 + 微服务思想。

```
                    +----------+
                    |  Nginx   | (反向代理 + 静态资源)
                    +----+-----+
                         |
         +---------------+---------------+
         |                               |
    +----v-----+                    +----v-----+
    | Vue3 SPA |                    | REST API |
    | Port:3000|                    | Port:8000|
    | (PC2开发) |                    | (PC3开发) |
    +----------+                    +----+-----+
                                        |
                    +-------------------+-------------------+
                    |                   |                   |
               +----v----+        +----v----+        +----v----+
               |PostgreSQL|        |  Redis  |        |DeepSeek |
               | +PGVector|        | Cache   |        |  V4 API |
               | Port:5432|        |Port:6379|        |(外部API) |
               +---------+        +---------+        +---------+

v4.0 审查链 (三闸门):
  PC3 代码提交 ──→ PC4 安全审查 (.171) ──→ PC5 QA 测试 (.253) ──→ PC1 终审 (.168)
                   第一闸门 (一级审查)      第二闸门 (二级审查)      第三闸门 (三级终审)
                   /cso 安全审计            /qa + /browse            Go/No-Go + 贝叶斯收敛判断
```

## 2. 技术选型

| 层面 | 技术 | 版本 | 选型理由 |
|------|------|------|----------|
| 后端框架 | Spring Cloud / Go-Fiber / FastAPI | 待定 | 自主选型 |
| ORM | GORM / SQLAlchemy / MyBatis-Plus | 对应框架 | 防SQL注入 |
| 数据库 | PostgreSQL + PGVector | 15+ | 向量检索+企业级 |
| 缓存 | Redis | 7.x | 高性能缓存+AI结果缓存 |
| 前端框架 | Vue3 + Vben5 | 3.x | 开箱即用后台模板 |
| 图表 | ECharts | 5.x | 决策链图谱可视化 |
| AI SDK | 兼容 OpenAI 格式 | latest | DeepSeek API兼容 |
| 工作流 | Flowable | 6.x | BPM流程引擎 |
| 认证 | JWT + OAuth2 | - | 无状态，适合分布式 |
