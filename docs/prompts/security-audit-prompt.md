# 安全审查 Prompt（PC4专用）

请对以下代码进行安全审查，重点检查：
1. SQL注入：是否使用参数化查询/ORM
2. XSS：用户输入是否转义
3. 认证授权：JWT验证/权限检查是否完整
4. 敏感数据：密码/Token/密钥是否加密存储
5. API安全：CORS配置/速率限制/输入校验
6. 日志安全：是否泄露敏感信息
