# 代码安全审查 Checklist (PC4一级审查标准)

## SQL注入
- [ ] 所有数据库查询使用参数化查询/ORM
- [ ] 无字符串拼接SQL
- [ ] 存储过程参数化

## XSS防护
- [ ] 所有用户输入输出进行HTML编码
- [ ] API响应Content-Type正确设置
- [ ] CSP头配置

## 认证与授权
- [ ] JWT Token过期时间合理（8小时）
- [ ] 密码使用bcrypt/scrypt加密
- [ ] RBAC权限检查在每个端点
- [ ] 无硬编码密钥/密码
- [ ] 连续失败锁定机制

## 数据安全
- [ ] 敏感字段加密存储（AES-256）
- [ ] HTTPS强制（生产环境）
- [ ] 日志不含敏感信息（密码/Token/身份证号）
- [ ] 文件上传限制类型和大小
- [ ] 操作日志保留90天可审计

## API安全
- [ ] 请求频率限制 (Rate Limiting)
- [ ] 输入参数校验（Pydantic/JSR303）
- [ ] CORS白名单（仅允许内网IP）
- [ ] 错误信息不泄露内部细节
- [ ] API版本管理

## 政企合规
- [ ] 数据本地存储（不离开局域网）
- [ ] 三重一大审批留痕
- [ ] 操作日志不可篡改
- [ ] 数据备份加密
